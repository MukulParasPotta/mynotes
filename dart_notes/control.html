Control Structures

Control Structures like:

  • if(cond){...} else if(cond){...} ... else{...},
  • for( ; ; ),
  • while(cond){...},
  • do{...}while(cond);,
  • switch(exp){case exp: ...},

work like in those in C Family of languages (C, C++, Java etc.)

Some built-in data structures:

  • [ ... ] is used to define List objects, which work like list from Python, which are iterable objects
  • {key: value, ... } is used to define Map objects, which work like dict from Python.

The above structures work like generics from Java, but type-inferred.

forEach looping

Let's look at an example.

        var list1 = ['A', 'B', 'C'];
        print(list1);

        list1.forEach(
            (item){
                print(item);
            }
        );
The above code gives the following output: 
['A', 'B', 'C']
A
B
C

Functions

Syntax like C Family, but supports nested functions like in Python. 

        void main(){
            var x = 10;
            void printIt(String name, String age){
                print(x);
                print(name);
                print(age);
            }
            printIt("Mukul", "20");
        }

If we change the signature to void printIt(String name, [String age]), age becomes an optional parameter. In that case,

        void main(){
            var x = 10;
            void printIt(String name, [String age]){
                print(x);
                print(name);
                print(age??x);
            }
            printIt("Mukul", "20");
        }

optional_param ?? default_value is one feature we can use to assign values to missing paramenters.

One-liner functions can be defined using Python-like syntax.
int isEven(int n) : return (n % 2) == 0;
