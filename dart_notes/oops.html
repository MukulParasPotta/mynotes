Functions

Functions outside classes can be defined like in C/C++.

One-liner functions can be defined using Python-like syntax.
int isEven(int n) : return (n % 2) == 0;

OOPS

Very similar to Java, except that there are no access modifiers like private, protected etc.

Following are similar to Java:

  • Constructor syntax, including the this keyword and its usage.
  • Class definition syntax, which includes attributes and member function definitions.
  • Inheritance and super keyword. 
    class SU extends U {SU() : super();}
  • Some generic classes like List<> are similar to those in Java.
