Data Types

All data types are inherited from Object class, no primitive types like Java.
  • var can be used to specify a variable of a yet-unknown type.
  • finalworks similar to var but cannot be modified later, needs initialization.
  • const is another keyword which works like final, but from conpile-time.
  • num is a class from which int, double are dervied.
  • String is the string class for Dart.
  • bool is the boolean data type.

obj.runtimeType gives the data type of the variable, which is an attribute from Object class.

obj.toString() converts all objects to String, if that method is defined.

Operators

Most operators work similar to Java, any exceptions will be listed here.

  • == no funky JS-like conversions, but works similar to that in Python.
  • + no String concatentaion overload like in Java.
  • ++, -- exist and work like in Java and can be used for looping, may also have a range class like in Python.
