Installation

Step 1:

Flutter needs JDK 8 so make sure it is installed and configured properly. 

 sudo apt-get install openjdk-8
Warning!!

The following link is for people with experience in Android development, and I recommend beginners to use Android Studio for developing Flutter apps, follow instructions here.

or follow this process to setup Flutter. This concludes all the pre-requisites.

Step 2:

Run

flutter doctor
to verify the installation.

Step 3:

Once that is done, you might have to run 

flutter doctor --android-licenses
to review and accept the licenses of the software used as part of Flutter. (think of it like accepting EULAs)
