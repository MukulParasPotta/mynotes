New Project

For my journey, I've used VS Code, but you can use any editor/IDE of your choice. But make sure it has Flutter (and Dart) plugins to make life easier.

Use Ctrl + Shift + P to call up the command line in VS Code (something similar exists for the IDE you're using, unless it doesn't support plugins), which lets you utilize the plugins installed.

Find Flutter commands, which must contain something like Flutter: New * Project. Here, * can be:

  • Application
  • Package
  • Module
  • Plugin
depending on your project, you can create any of the above, but I proceeded with an Application.

Contents of a Project

Running Flutter: New Application Project writes a lot of files which is essentially a sample application.

There's a lot of folders and files in there, but to use Flutter, only few are actually used (read as relevant for beginners).
